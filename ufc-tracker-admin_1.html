Firebase Setup Instructions

To enable real-time syncing, you need to set up Firebase (it's free!):

  1. Go to Firebase Console
  2. Click "Add project" and create a new project
  3. Click the web icon (</>) to add a web app
  4. Copy your Firebase config object
  5. Paste it in the box below
  6. Go to "Realtime Database" in Firebase Console
  7. Click "Create Database" and start in test mode

Paste Your Firebase Config

It should look like this:

{
  apiKey: "AIza...",
  authDomain: "your-app.firebaseapp.com",
  databaseURL: "https://your-app.firebaseio.com",
  projectId: "your-project",
  storageBucket: "your-app.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123:web:abc123"
}
